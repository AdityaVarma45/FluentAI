# FluentAI ✨

🚀 **GenAI-powered English learning platform** built using modern AI models (Groq) to improve grammar, writing, and communication skills with real-time intelligent feedback.

🔗 Live: https://fluentai-4z4w.onrender.com  
📦 Repo: https://github.com/AdityaVarma45/FluentAI  

---

## 🤖 GenAI Overview

FluentAI leverages **Generative AI (GenAI)** to analyze, enhance, and evaluate user input in real time.

Using AI models via **Groq API**, the platform provides:
- Context-aware grammar corrections  
- Intelligent sentence rewriting  
- Structured essay evaluation  
- Meaningful summarization  

This transforms traditional learning into an **interactive AI-driven experience**.

---

## 🎯 Features

- 🤖 **AI Grammar Checker** – Detects and corrects grammar with explanations  
- ✍️ **AI Rewrite Tool** – Improves clarity and professionalism  
- 📄 **AI Essay Evaluator** – Provides structured feedback like IELTS examiners  
- 📚 **AI Vocabulary Boost** – Generates useful words with meanings  
- 📌 Bookmark AI responses  
- 🕓 Track full history of AI interactions  
- 🔥 Learning streak tracking  
- ⚡ Real-time AI responses with typing effect

## 🧠 Architecture
Frontend → /api → Backend → GenAI (Groq) → Database
- Frontend (React) handles UI and user interactions  
- Backend (Express) manages API requests and AI processing  
- GenAI (Groq) performs real-time text analysis and generation  
- MongoDB stores user data, history, and bookmarks  

- Single-origin production setup  
- Vite build served via Express  
- Clean separation between UI, API, and AI layers-
